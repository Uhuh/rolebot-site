About RoleBot

RoleBot is a reaction role Discord bot designed to make it easy to categorize your react roles and allow your members to select the roles they want.

RoleBot was started in September 2018 as a goofy project for a single server but eventually enough members asked if they could use RoleBot that it inspired me to make it public.

If you use RoleBot and want to support the development you can buy me a coffee!
