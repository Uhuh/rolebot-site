Frequently asked questions.

How do I invite RoleBot?

You can invite RoleBot here.

Can I selfhost RoleBot?

Yes you can selfhost RoleBot if you're a little tech savy. Visit the GitHub here.

I ran /react-channel but RoleBot didn't send any embeds.

This is usually due to RoleBot not having the correct permissions for the channel you want the embeds in.

I ran /help and nothing happened.

RoleBot uses Discords slashcommands. This means that RoleBot needs to be invited with the ability to "create slash commands". The correct invite link is in this FAQ, top right, or in the footer. Invite RoleBot from any of those!

I set up RoleBot but it's not giving users roles.

Due to how Discord manage roles works, RoleBot's role or any role you give Rolebot needs to be at minimum be above all roles you want RoleBot to give out.
